# Projeto Relatório de Produtos

Este projeto foi desenvolvido utilizando a **IDE Lazarus** em conjunto com o **Fortes Report**, uma poderosa biblioteca de relatórios para aplicações em Pascal/Delphi.  
O objetivo é gerar relatórios de produtos de forma prática e organizada.

## Tecnologias utilizadas
- **Lazarus** (IDE para desenvolvimento em Pascal/Free Pascal)
- **Fortes Report** (ferramenta de geração de relatórios)

## Funcionalidade principal
O sistema gera um relatório de produtos contendo os seguintes campos:

- **Código**  
- **CEAN** (Código de barras)  
- **Descrição**  
- **Estoque**  
- **Preço de venda**

## Estrutura do projeto
- Código-fonte em **Pascal**
- Relatórios configurados no **Fortes Report**
- Saída em formato impresso ou exportável (PDF/visualização)

## Como executar
1. Abra o projeto no **Lazarus**.
2. Certifique-se de que o **Fortes Report** está instalado e configurado.
3. Compile e execute a aplicação.
4. Gere o relatório de produtos através da interface.

## Observações
- Este projeto é voltado para fins de estudo e prática com Lazarus e Fortes Report.
- Os relatórios podem ser adaptados para incluir novos campos ou filtros conforme necessidade.
